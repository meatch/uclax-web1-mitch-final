Q1. Explain the difference between HTML, CSS and JS.

  1. HTML: Content: Structure of the Webpage - tagged elements, open and closed tag - most human readable
  2. CSS: Appearance: The visual Layer (The Human Skin)
  3. JS: Behavior: Interactivity to the website, validating form, remembering the user, altering content
